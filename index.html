Currently very interested in non-euclidian level design in games.

VR Terrain Sculpting Tool

This is a project that is in progress, but it works. It is a terrain sculpting tool designed for game developers. VR is always being used by consumers for gaming, but what if it could be useful in the development process? My friend Alaina and I wondered this and set out to make a level editor in VR. Its pretty neat and you get to shoot lasers from your hands. You can find it here.
